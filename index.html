VS Code ショートカット

操作概要 macOS
全般
すべてのコマンドの表示 ⇧⌘P, F1
クイックオープン ⌘P
キーボード ショートカットを開く ⌘K ⌘S
直近で開いたファイルを表示 ⌃Tab
UI全体の拡大 ⌘S+
UI全体の縮小 ⌘+
編集
コードの整形 ⇧⌥F
（選択範囲なしで）行の切り取り ⌘X
（選択範囲なしで）行のコピー ⌘C
行を下へ移動 ⌥↓
行を上へ移動 ⌥↑
行を下へコピー ⇧⌥↓
行を上へコピー ⇧⌥↑
行の削除 ⇧⌘K
行を下に挿入 ⌘Enter
行のインデント追加 ⌘]
行のインデント削除 ⌘[
名前を変更 F2
補完候補を開く ⌃Space
選択
矩形選択(マウス選択) ⇧⌥+ドラッグ
矩形選択(キーボード選択) ⇧⌥⌘+方向キー
選択範囲と同じ文字列の複数指定 ⌘D
移動
行の先頭に移動 Home
行の最後に移動 End
ファイルの先頭に移動 ⌘↑
ファイルの末尾に移動 ⌘↓
行コメントの切り替え ⌘/
指定行へ移動 ⌃G
ファイルに移動 ⌘P
前に戻る ⌃-
次に進む ⌃⇧-
検索
ファイル内検索 ⌘F
ファイル内置換 ⌥⌘F
次を検索 ⌘G
前を検索 ⇧⌘G
プロジェクト内検索 ⇧⌘F
パネル操作
サイドバーの表示/非表示 ⌘B
ターミナルの表示/非表示 ⌃@
新しく統合ターミナルを表示 ⌃⇧@
エクスプローラーの表示 ⇧⌘E
参考：Visual Studio Code キーボード ショートカット
